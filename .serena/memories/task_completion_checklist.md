# Task Completion Checklist

Before announcing task completion, verify:

## Code Quality
- [ ] Tests written (TDD preferred)
- [ ] Tests pass: `zig build test`
- [ ] Code formatted: `zig fmt src/`
- [ ] Build succeeds: `zig build`

## Documentation
- [ ] Update kanban task card per `kanban/CLAUDE.md`
- [ ] Leave progress notes

## Design
- [ ] Check coupling/cohesion
- [ ] Intent-revealing naming
- [ ] No unnecessary code added
