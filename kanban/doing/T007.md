## Handover Notes

### Spread Algorithm (sine wave with anchored edges)
The key formula in `cardRects()`:
```zig
// t = normalized card position with margin [0.1, 0.9]
// m = normalized mouse position [0, 1]
const weight: f32 = if (t < m and m > 0.001)
    -@sin(std.math.pi * t / m)        // left of mouse: push left
else if (t > m and m < 0.999)
    @sin(std.math.pi * (t - m) / (1.0 - m))  // right: push right
else
    0;
card_x = base_x + weight * max_displacement;
```

### Next Steps for Rotation
1. In `graphics.zig`, update `renderCard()` to use SDL's rotated texture rendering
2. In `CarouselView.appendRenderables()`, calculate rotation based on weight:
   - `rotation = weight * max_angle` (e.g., max_angle = 15 degrees)
   - Cards left of mouse tilt left (negative), right tilt right (positive)
   - Card under mouse is vertical (weight=0)
